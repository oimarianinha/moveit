## Instruções para uso

Clone o projeto e vá até a pasta

```bash
$ git clone https://github.com/oimarianinha/moveit && cd moveit
```

Siga os passos abaixo
```bash
# Instale as dependências
$ yarn

# Inicie o projeto
$ yarn dev
```
